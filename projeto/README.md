# Introdução ao Desenvolvimento de Sistemas de Software (2026)
Este repositório contém tarefas de casa/sala e projeto individual, referentes
à disciplina MAC0350, do quinto semestre do curso de Ciência da Computação (IME-USP).
Abaixo está um esboço do projeto individual, que está em fase de desenvolvimento.

## Casa de Malhas

### Descrição

Site da empresa fictícia *Casa de Malhas*, referente ao projeto pessoal 
da disciplina de Introdução ao Desenvolvimento de Sistemas de Software.

O site deve estar compartimentado em diferentes seções: 
+ Endereço e contato (email e telefone)
+ Vitrine virtual: alguns produtos da loja
+ Formulário para insesão do CEP e cálculo do frete
+ Formulário para insersão de cupom de desconto
+ Sistema de login de usuário
+ Compra de produtos: formulário com a peça selecionada para entrega

É importante que o site esteja razoavelmente estilizado, afim de simular
um site de uma empresa real.

Além disso, vale destacar que o projeto ainda está "no papel", portanto,
é possível que novas ideias possam surgir até a entrega do trabalho.
